# 🩺 Diabetes Predictor Web App

A machine learning web application that predicts whether a patient 
is diabetic or not based on medical parameters.

## 🔴 Live Demo
👉 [Try the app here](https://diabetes-predictor-9971.onrender.com)

## 📌 About The Project
This project uses a Machine Learning model trained on the 
Pima Indians Diabetes Dataset to predict diabetes risk in patients.
The model analyzes 8 medical parameters and predicts whether 
a patient is diabetic or not.

## 🛠️ Tech Stack
- **Python** — Core programming language
- **Scikit-learn** — ML model training
- **Pandas & Numpy** — Data processing
- **Flask** — Web framework
- **HTML/CSS** — Frontend
- **Render** — Deployment

## 📊 Dataset
- **Source:** Pima Indians Diabetes Dataset (Kaggle)
- **Size:** 768 patients
- **Features:** 8 medical parameters

## 🔍 Features Used For Prediction
| Feature | Description |
|---|---|
| Pregnancies | Number of pregnancies |
| Glucose | Glucose level |
| Blood Pressure | Blood pressure value |
| Skin Thickness | Skin thickness value |
| Insulin | Insulin level |
| BMI | Body mass index |
| Diabetes Pedigree Function | Diabetes hereditary score |
| Age | Patient age |

## 🤖 Model Performance
- **Algorithm:** Logistic Regression
- **Accuracy:** 76%
- **Dataset Split:** 80% training, 20% testing
